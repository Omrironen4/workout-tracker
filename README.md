# workout-tracker
 - Create a new exercise 
 - Add a new exercise 
 - View the stats of the exercises such as duration and weight across a seven day graph.

## Link to deployed app
https://oms-workout-tracker.herokuapp.com/


## Technologies Used

- JavaScript 
- Node.js
- MongoDB
- Mongoose
- Morgan
- Express
